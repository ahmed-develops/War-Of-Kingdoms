版本号更新流程

步骤1：每次打开先更新一下SVN

步骤2：打开database.local.xml，将前端修改的数据同步到此配置中（此文件为程序用前台数据总表，前台数据必须更新到此文件中才可生效）

步骤3：将修改完毕的database.local.xml以及text_zh_CN.ini一起选中压缩成一个ZIP的文件包（压缩包的版本号末尾数字变更一下，变更规则：如果原来是04现在就改成05，并将老的文件包删除）

步骤4：打开\xml\config.txt文件最后的末尾数字变更一下（变更规则：如果原来是04现在就改成05）

步骤5：将所有修改内容提交，写明注释以便复查

步骤6：打开CRT连接
