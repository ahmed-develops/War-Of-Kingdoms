Methods Available

There are 3 different methods available to install LegacyClash - CoC on your Android device. Check them out, no matter if you are rooted or not!

More Info

Client Downloader

NO ROOT REQUIRED

This is the best method - Our Client Downloader will grab the application for you, so you can connect to us without root.

Let's Go APP

DNS

NO ROOT REQUIRED

DNS method does not require root. This method works only if you are connected to a Wi-Fi network. You can't play on our server using this method if you are using mobile data.

Let's Go DNS

IP

ROOT REQUIRED

IP method works only if you have a rooted device, because you will have to access hosts file which is a system file and can't be accessed without root. You can play on our server with this method even if you are not connected to a Wi-Fi network (Internet Access is necessary), and you can use mobile data too.
